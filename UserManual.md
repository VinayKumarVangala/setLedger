## üéØ Getting Started

### First Time Setup

#### 1. Organization Registration
1. **Visit Application**: Open setLedger in your browser
2. **Create Organization**: Click "Create New Organization"
3. **Fill Details**:
   - Organization Name
   - Admin Email & Password
   - GST Number (optional)
   - Business Type
4. **Verify Email**: Check email for verification link
5. **Complete Setup**: Follow onboarding wizard

#### 2. User Roles Overview
- **Admin**: Full system access, user management, security settings
- **Accountant**: Financial operations, GST filing, reports
- **Analyst**: View reports, analytics, forecasting
- **Staff**: Basic operations, POS, inventory updates
